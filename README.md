## 使用方法
1. Clone 本仓库源码 & 在 Release 页下载
2. 获取你的爱发电 UserId 和 API Token [点此一键直达](https://afdian.net/dashboard/dev)
3. 将这些信息按照注释填入 `index.php` 文件头部
4. 填写其他信息
5. 上传至任何支持 PHP 的服务器/虚拟主机/网页托管平台 etc.
6. 访问一下试试吧！
